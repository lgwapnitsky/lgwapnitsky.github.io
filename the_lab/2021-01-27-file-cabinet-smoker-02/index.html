The File Cabinet Smoker Project - Part II - More Microelectronics

Now that I had some basic knowledge, it was time to work on figuring out how to turn a drawing into something that worked - theoretically.

Back in Part I, I discussed my plans to build an Arduino microelectronic-controlled smoker, and the start of how I had to learn to even use and program the microcontroller itself. TinkerCAD was my savior, and I started to really get comfortable with it, so now was the time to see if I could interpret the schematic designed by Charles Hartley into something functional.

Again, let me state that I have NO training in reading, interpreting, or creating electrical schematic diagrams, so this was a new challenge. My background is in programming, so understanding the code was easy. Interpreting all these lines and symbols, not so much. Thankfully, I had some knowledge of basic icons for resistors and LEDs, and the provided schematic was very detailed in how the wiring should go. Everything else, like how to hook-up the breadboard prototype, was Greek to me.
